Super Simple SpringBoot Board Project
Practice Web MVC, CRUD

Tech Skills

Springboot
의존관계 주입(DI)을 통해 쉽게 객체 관계 구성을 도와주는 것을 활용하여 객체 지향 프로그래밍을 하기 위해 사용했습니다.
또한 MVC 구조를 편리하게 개발하기 위해 사용했습니다.
MariaDB
명확한 데이터 구조를 보장받고 테이블간의 관계에서 외래 키를 이용한 JOIN을 활용하기 위해 사용했습니다.
Mybatis
조건에 따른 동적 쿼리 작성과 쿼리 재사용성을 활용하기 위해서 사용했습니다.
HTML
어떤 운영체제에서도 웹브라우저만 있으면 볼 수 있어 접근성이 좋기때문에 사용했습니다.
CSS
HTML의 스타일을 관리하기 위해 사용했습니다.
jQuery
이벤트가 발생하면 DOM을 선택하고 DOM의 값을 변경하기 위해 jQuery를 사용했습니다.
