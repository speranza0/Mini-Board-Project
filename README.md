# SpringBoot-Project-MiniBoard
스프링 부트 + 타임리프 게시판

## 🖥️ 프로젝트 소개
게시글 등록 수정 삭제 (CRUD)를 구현한 MVC 구조의 게시판 프로젝트입니다.

## 🕰️ 개발 기간
22.10.12일 - 22.11.08일

### ⚙️ 개발 환경
- `Java 11`
- `JDK 11.0.15`
- **IDE** : IntelliJ
- **Framework** : Springboot(2.x)
- **Database** : MariaDB
- **ORM** : Mybatis

## 📌 주요 기능
#### 로그인
- 로그인 시 세션(Session) 생성
- 관리자 계정(테스트용) 로그인
- 일반회원 계정(테스트용) 로그인
